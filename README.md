# Лабораторная работа по базам данных

Проект для выполнения лабораторной работы по базам данных с использованием Node.js и PostgreSQL.

## Установка

1. Убедитесь, что у вас установлен Node.js и PostgreSQL
2. Скопируйте файлы проекта
3. Установите зависимости:
   ```bash
   npm install
   ```

## Настройка

Создайте файл `.env` в корне проекта (уже создан в проекте) и укажите параметры подключения к базе данных:

```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=lab_db
DB_USER=postgres
DB_PASSWORD=postgres
```

## Инициализация базы данных

Для создания базы данных и таблиц выполните:

```bash
npm run init-db
```

## Запуск

Для запуска основного приложения:

```bash
npm start
```

## Структура проекта

- `index.js` - основной файл приложения
- `init-db.js` - скрипт для инициализации базы данных
- `package.json` - зависимости и скрипты проекта
- `.env` - переменные окружения для подключения к базе данных
